 # Node.js + Express Learning Journey 🚀

This repository documents my progress while completing the **Node.js + Express course**.  
I explored backend fundamentals, API development, and practical workflows with Express.

##  Key Learnings
- Listening to requests and sending responses
- Passing data through routes and middleware
- Handling query strings in Express
- Using Nodemon for auto-restarts during development
- Structuring APIs for scalability
